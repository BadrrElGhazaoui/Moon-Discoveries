# 🌙 Moon Discoveries — Personal Blog

> *"Like owning a house makes you feel secure, I own this corner of the internet."*

A personal blog built with [Beautiful Jekyll](https://beautifuljekyll.com/), hosted on GitHub Pages. I write about the things that move me — songs, movies, books, and the occasional tech topic.

📖 **Live blog → [badrrelghazaoui.github.io](https://badrrelghazaoui.github.io/)**

---

## About

Hi, I'm **Badr El Ghazaoui**, a 19-year-old from Casablanca, Morocco.

I started this blog for a few simple reasons:
- I love books and want to share what I take from them
- Movies deserve more than a dopamine hit — I want to dig into what they mean
- I have some IT knowledge worth putting out there
- And mostly, for myself
